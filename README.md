# PersonalTrainerBlog
Personal Traner Website with wysiwyg editor

www.katyaperrott.herokuapp.com

Features:
- Responsive design
- Message form
- Multilanguage site (under testing)
- WYSIWYG editor
- Authentication with bcrypt hashing

Front-End
- Bootstrap 4.1.3
- HTML5
- JavaScript
- jQuery
- CSS3

Back-End:
- Node.js 
   +Handlerbars
   +express
   +exports
   +path
   +body-parser
   +nodemailer
   +nodemailer-smtp-transport
   +SQLlite
   +pasport
   +sessions
   +cookies
